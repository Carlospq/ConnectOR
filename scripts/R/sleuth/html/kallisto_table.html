kallisto_table {sleuth}R Documentation

Get kallisto abundance table from a sleuth object

Description

Get back the kallisto abundance table from a sleuth object.

Usage

kallisto_table(obj, use_filtered = FALSE, normalized = TRUE,
  include_covariates = TRUE)

Arguments

obj

a sleuth object

use_filtered

if TRUE, return the filtered data

normalized

if TRUE, return the normalized data. Otherwise, return the raw data

include_covariates

if TRUE, join the covariates

Value

a data.frame with at least columns
