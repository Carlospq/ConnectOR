subset_kallisto {sleuth}R Documentation

Subset a kallisto object

Description

Use a vector of target_ids to subset a single kallisto object or all of kallisto objects in a sleuth object.

Usage

subset_kallisto(obj, ...)

Arguments

obj

either a sleuth or kallisto object

Value

a new object with only the subset of target_ids included
