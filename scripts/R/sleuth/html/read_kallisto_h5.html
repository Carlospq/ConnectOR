read_kallisto_h5 {sleuth}R Documentation

Read a kallisto object from an HDF5 file

Description

Read a kallisto object from an HDF5 file.

Usage

read_kallisto_h5(fname, read_bootstrap = TRUE, max_bootstrap = NULL)

Arguments

fname

the file name for the HDF5 file

read_bootstrap

if TRUE load bootstraps, otherwise do not

max_bootstrap

an integer denoting the number of bootstraps to read. if NULL read everything available

Value

a kallisto object
